Справка к набору скриптов daek

Какие проблемы это решает:
    1. Преобразование csv файла в json 
        При этом подразумевается что исходный файл, находится в кодировке utf-8 with BOM (Microsoft Excel) 
        csv2json путь_к_файлу.csv путь_к_файлу.json название_таблицы разделитель, требуется заранее убедиться в отустствии лишних ; в конце строки

        результатом работы будет однострочный json

    2. Преобразование однострочного JSON в полноценный удобный для чтения формат 
        jsoneyelike путь_к_файлу
        При этом оригинальный файл будет заменен 

    3. Разрезание болшого JSON файла на множестов мелких
        Для разрезания годятся только файлы пропущенные через jsoneyelike
        jsonax ()
